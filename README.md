# Red Team Simulation (Angular)

A standalone Angular + Angular Material + Tailwind decision-tree simulation game with a hacker-style UI.

## Quick Start

```bash
npm install
npm start
```

Open `http://localhost:4200`.

## Features
- Branching scenario engine loaded from `assets/data/game-data.json`
- Save/Load progress (LocalStorage)
- Hint system (3 hints)
- Keyboard shortcuts `1-4`
- Export path as Markdown
- Victory/failure debrief screens
